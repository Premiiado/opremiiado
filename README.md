## Instalar
Siga os passos abaixo!

```bash
> termux-setup-storage
(depois disso toque na permissão)
> apt install git
> pkg install ffmpeg
> apt install wget
> pkg install nodejs
> pkg install npm
> git clone https://github.com/premiiado/opremiiado.git
> cd opremiiado
> bash install.sh
```

## Uso

```bash
> npm start
> leia o códico QR e após isso de um exit
> cd ptzeus
> npm i -g pm2
> pm2 start index.js
> pm2 monit
```
